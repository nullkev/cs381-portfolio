# [AS6]
> Pokemon styled combat using characters, models, voicelines, and names from the game Azur Lane.

## Introduction
During a sortie, the commander and New Jersey encounter a Prinz Eugen and she challenges you to a fight. You must command New Jersey to victory. 

Reduce Prinz Eugen's HP to zero to win the fight!

## Getting Started

### Building
    $ git submodule init
    $ git "submodule update --init --recursive
    $ mkdir build
    $ cd build
    $ cmake..
    $ make
    $ ./as6

### Usage
##### Controls
###### Titlescreen Controls
    <<< ENTER >>> = move from title screen to battle.
###### Battle Controls
    <<< 1 >>> = select and use New Jersey's first move.
    <<< 2 >>> = select and use New Jersey's second move.
    <<< 3 >>> = select and use New Jersey's third move.
